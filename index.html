Backend

E-commerce

This is the backend of my e-commerce project. In this project, I have used mongoDB as my open source database management. Other tools that I have used is JavaScript, node.js and express.js. In this project, a registration feature is available and there are two types of users which is the admin and regular user. I was able to separate the access of the two users by using the jsonwebtoken authentication. I also used Models, Controllers and Routes in order to manage my project effectively. In this project, the regular user should be able to login, browse available products, create an order or add the selected product to their cart and view order history. As for the admin, they are able to change another user to an admin, change the product information and view all order history.
